# ACF / SCF Custom Field Type – Lucide Icons

Adds a custom field type **“Lucide Icons”** to **Advanced Custom Fields (ACF)** and **Secure Custom Fields (SCF)**, allowing you to easily select and use Lucide icons in WordPress.

SCF is a community-driven fork of ACF, and this plugin is compatible with both.

## Features

- Custom field type for ACF and SCF
- Uses the Lucide icon set (SVG-based)
- Icon picker directly in the field settings
- Works with ACF (Free) and SCF
- Developer-friendly and extendable
- Suitable for commercial and non-commercial projects

## Requirements

- WordPress
- Advanced Custom Fields (ACF) **or**
- Secure Custom Fields (SCF)

> ⚠️ This plugin does **not** bundle ACF or SCF.  
> One of them must be installed and activated separately.

## Installation

1. Download or clone this repository
2. Upload the plugin folder to `/wp-content/plugins/`
3. Activate the plugin in the WordPress admin
4. Create or edit an ACF / SCF Field Group
5. Add a field of type **“Lucide Icons”**

## Usage

Once added, the field allows you to select a Lucide icon.

The selected value can be used in your theme or plugin, for example to render the SVG icon on the frontend.

Implementation details depend on your setup and rendering preferences.

## Compatibility

- ✔ Advanced Custom Fields (ACF)
- ✔ Secure Custom Fields (SCF)

This plugin relies only on public APIs and hooks provided by ACF / SCF and does not include or modify their source code.
